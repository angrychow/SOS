| switch(number) | syscall                | 功能简介                                               |
|----------------|------------------------|------------------------------------------------------|
| 3              | open                   | 打开文件并返回文件描述符                                 |
| 4              | close                  | 关闭文件                                               |
| 5              | read                   | 从文件中读取数据                                       |
| 6              | writes                 | 向文件中写入数据                                       |
| 7              | delete                 | 删除文件                                               |
| 8              | mkdir                  | 创建目录                                               |
| 9              | create                 | 创建文件                                               |
| 10             | -                      | -                                                    |
| 11             | read shared memory     | 读取共享内存中的数据                                   |
| 12             | write shared memory    | 向共享内存中写入数据                                   |
| 13             | 释放信号量             | 释放指定的信号量                                       |
